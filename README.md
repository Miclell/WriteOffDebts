# WriteOffDebts

WriteOffDebts - это мем-сайт для управления долгами, созданный с использованием ASP.NET Core MVC 8 и Entity Framework. Сайт позволяет пользователям добавлять своих должников и начислять проценты по долгам. Кроме того, сайт поддерживает фиктивную оплату, которая позволяет "оплачивать" долги без реального списания денег.

## Функциональность

- Добавление должников с указанием суммы долга и даты возникновения долга
- Просмотр списка своих должников
- Начисление процентов по долгам
- Фиктивная оплата долгов с удалением должника из списка
- Регистрация пользователей

## Технологии

- ASP.NET Core MVC 8
- Entity Framework
- jQuery

## Установка и запуск

1. Скопируйте репозиторий на свой компьютер, выполнив команду `git clone https://github.com/your_username/writeoffdebts.git`.
2. Откройте решение в Visual Studio.
3. Восстановите пакеты NuGet, выполнив команду `dotnet restore` в терминале.
4. Создайте базу данных, выполнив миграции Entity Framework, выполнив команду `dotnet ef database update` в терминале.
5. Запустите приложение, нажав кнопку F5 в Visual Studio.
